BioModelsRAG-website

CAUTION!!: This tool has been sized down to be able to run on the web - therefore, it has limited functionalities and it is recommended that the user download and use the Python package. The website also uses a lower level of Llama (llama-2 rather than llama-3 compared to the Python package. You can also download and use the source code to customize the Llama model configuration

It can be installed using the following command from any terminal with a Python virtual environment set up. Please refer to the Github pages for more information.

pip install biomodelsrag
